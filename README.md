# Learning-CSharp-Calc

C#の基礎学習を目的とした、コンソール上で動作するシンプルな加算アプリケーションです。標準入力からのデータ取得と、数値型へのキャスト、変数演算の基本を実装しています。

## 概要
ユーザーから2つの整数を入力として受け取り、その合計値を計算してコンソール画面に表示します。

## 開発環境
* 言語: C#
* ランタイム: .NET 6.0 以降（推奨）
* IDE: Visual Studio 2022 / Visual Studio Code

## ファイル構成
本リポジトリにはソースコードおよびプロジェクトファイルのみを格納しています。
* Program.cs: アプリケーションのメインロジック。
* Learning-CSharp-Calc.csproj: プロジェクトの構成情報。

※ビルド済みの実行バイナリは、リポジトリ右側の「Releases」セクションから取得可能です。

## 実行・ビルド方法

### 1. ソースから実行する場合
.NET SDKがインストールされている環境にて、プロジェクトのルートディレクトリで以下のコマンドを実行してください。
dotnet run

### 2. ビルドする場合
実行ファイルを生成するには以下のコマンドを使用します。
dotnet build

## 学習項目
このプロジェクトでは以下のC#機能の習得を目的としています。
* Console.WriteLine() による標準出力。
* Console.ReadLine() による標準入力の取得。
* int.Parse() による文字列から整数型への変換。
* $""（文字列補完）を用いた変数の埋め込み出力。
